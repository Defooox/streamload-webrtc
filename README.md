# streamload-webrtc
Разработать систему для загрузки и просмотра файлов через браузер с использованием WebRTC для потоковой передачи.

Техническое задание:

Подготовка среды (1–2 дня)

Сборка libwebrtc из исходников (5–10 дней)

Поднять и запустить один из примеров (10–15 дней)

MVP функционала загрузки + синхронного просмотра (7–12 дней)

UI/UX и обёртка (оставшееся время)
